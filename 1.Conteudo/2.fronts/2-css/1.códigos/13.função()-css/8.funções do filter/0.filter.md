# **`filter`** É um **atributo** usado para **aplicar efeitos visuais** em elementos.

- `blur()` → desfoca
    
- `brightness()` → ajusta o brilho
    
- `contrast()` → ajusta o contraste
    
- `grayscale()` → escala de cinza
    
- `sepia()` → efeito sépia
    
- `invert()` → inverte cores
    
- `opacity()` → transparência
    
- `drop-shadow()` → sombra
