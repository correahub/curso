# `linear-gradient()` é uma **função  CSS** que cria uma transição suave entre duas ou mais cores em uma **direção reta**.


**Exemplo em CSS:**

```css
.box {
  background: linear-gradient(to right, red, blue);
}
```

➡️ Nesse exemplo, a cor vai **do vermelho para o azul**, da **esquerda para a direita**.

**Outro exemplo (com ângulo):**

```css
background: linear-gradient(45deg, yellow, orange, red);
```

💡 Muito usada em `background` para criar efeitos visuais modernos sem imagens.
