# **`border-collapse`** é um **atributo** usado em tabelas para **definir se as bordas das células serão separadas ou colapsadas em uma só**.

Exemplo:

```html
<table style="border-collapse: collapse; border: 1px solid black;">
  <tr>
    <td style="border: 1px solid black;">Caneta</td>
    <td style="border: 1px solid black;">R$ 2,50</td>
  </tr>
</table>
```
