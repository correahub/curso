| Tipo                   | Significado (exemplos)                                        |
| ---------------------- | ------------------------------------------------------------- |
| `<number>`             | números ( `0`, `1`, `2.5` )                                   |
| `<integer>`            | números inteiros ( `1`, `2`, `10` )                           |
| `<length>`             | tamanhos ( `10px`, `2rem`, `50vh` )                           |
| `<length-percentage>`  | tamanho ou porcentagem ( `20px`, `50%` )                      |
| `<percentage>`         | porcentagem ( `0%`, `50%`, `100%` )                           |
| `<color>`              | cores ( `red`, `#ff0000`, `rgb(255,0,0)` )                    |
| `<angle>`              | ângulos ( `45deg`, `1turn`, `3.14rad` )                       |
| `<time>`               | tempo ( `300ms`, `1s` )                                       |
| `<resolution>`         | resolução ( `96dpi`, `2dppx` )                                |
| `<ratio>`              | proporção ( `16/9`, `4/3` )                                   |
| `<position>`           | posição ( `top`, `center`, `left` )                           |
| `<image>`              | imagens/gradientes ( `url(img.png)`, `linear-gradient(...)` ) |
| `<url>`                | endereço de arquivo ( `url("img.png")` )                      |
| `<transform-function>` | transformações ( `rotate(45deg)`, `scale(1.2)` )              |
| `<string>`             | texto ( `"abc"`, `"titulo"` )                                 |
| `<custom-ident>`       | identificadores ( `auto`, `primary` )                         |
| `*`                    | qualquer valor ( `10px`, `red`, `rotate(45deg)` )             |
