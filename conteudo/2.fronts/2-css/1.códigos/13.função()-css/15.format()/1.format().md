# `format()` é uma **função do CSS** que informa ao navegador **qual é o formato do arquivo de fonte**, ajudando na compatibilidade.

**Exemplo:**

```css
@font-face {
  font-family: "MinhaFonte";
  src: url("fonte.woff2") format("woff2");
}
```

➡️ O navegador sabe exatamente **como interpretar o arquivo**.
