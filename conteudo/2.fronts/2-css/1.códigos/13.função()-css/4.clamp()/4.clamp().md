# **`clamp()`** é uma função CSS usada para **limitar um valor dentro de um intervalo, definindo mínimo, preferencial e máximo**.

Exemplo:

```css
div {
  font-size: clamp(14px, 2vw, 20px);
}
```

Nesse caso:

- O **mínimo** é 14px
    
- O **preferencial** é 2vw (2% da largura da viewport)
    
- O **máximo** é 20px
    

Ou seja, o valor **nunca será menor que 14px nem maior que 20px**, ajustando dinamicamente dentro do intervalo.
