# **`rgba()`** é uma função do CSS usada para **definir cores combinando vermelho, verde e azul, com transparência (alfa)**.

Exemplo:

```css
div {
  background-color: rgba(255, 0, 0, 0.5); /* vermelho semi-transparente */
}
```

O último valor (0.5) define a **opacidade**, variando de 0 (totalmente transparente) a 1 (totalmente opaco).
