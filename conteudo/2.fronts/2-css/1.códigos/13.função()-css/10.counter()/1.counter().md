# **`counter()`** é uma função do CSS usada para **exibir o valor atual do `counter-reset`.


Exemplo:

```css
body {
  counter-reset: seção;
}

h2::before {
  counter-increment: seção;
  content: "Seção " counter(seção) ": ";
}
```

Aqui, cada `<h2>` exibirá **"Seção 1: ", "Seção 2: "**, etc., automaticamente.
