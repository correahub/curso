
# `local()` é uma **função  CSS** usada em `@font-face` para **carregar uma fonte que já está instalada no sistema do usuário**, sem baixar arquivo.

**Exemplo:**

```css
@font-face {
  font-family: "MinhaFonte";
  src: local("Arial");
}
```

➡️ O navegador usa a **fonte local** se ela existir no computador.
