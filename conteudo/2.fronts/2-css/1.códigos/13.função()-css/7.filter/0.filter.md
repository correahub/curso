# **`filter é o atritubo que`** modifica a aparência visual de um elemento usando efeitos gráficos.

- `blur()` → desfoca.
		**Valor mínimo:** `0px` → sem desfoque (imagem totalmente clara).  
		**Valor máximo:** **não possui limite oficial**, mas valores muito altos (ex: `50px`, `100px`) deixam o elemento praticamente irreconhecível.
    
- `brightness()` → ajusta o brilho.
		**Valor mínimo:** `0%` ou `0` → elemento totalmente preto.  
		**Valor padrão:** `100%` ou `1` → brilho normal.  
		
    
- `contrast()` → ajusta o contraste.
		**Valor mínimo:** `0` → remove todo o contraste, deixando a imagem completamente cinza.  
		**Valor padrão:** `1` ou `100%` → contraste normal.  
		**
- `grayscale()` → escala de cinza
		**Valor mínimo:** `0` → sem efeito (cores normais).  
		**Valor máximo:** `1` ou `100%` → totalmente em preto e branco.
    
- `sepia()` → efeito sépia
		**Valor mínimo:** `0` → sem efeito (cores originais).  
		**Valor máximo:** `1` ou `100%` → efeito sépia total.
    
- `invert()` → inverte cores
		**Valor mínimo:** `0` → sem inversão (cores normais).  
		**Valor máximo:** `1` ou `100%` → cores totalmente invertidas.
    
- `opacity()` → transparência
		**Valor mínimo:** `0` → totalmente transparente (invisível).  
		**Valor máximo:** `1` ou `100%` → totalmente visível (sem transparência).
    
- `drop-shadow()` → sombra
	- - **offset-x** → deslocamento horizontal
	    
	- **offset-y** → deslocamento vertical
	    
	- **blur-radius (opcional)** → nível de desfoque da sombra
	    
	- **color (opcional)** → cor da sombra
