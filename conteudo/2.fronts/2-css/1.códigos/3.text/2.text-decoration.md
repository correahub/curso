# **`text-decoration`** é o **atributo** que define **decorações aplicadas ao texto**.




















### 1. `text-decoration-line`Define **onde a linha aparece no texto**.

```css
text-decoration-line: underline; /* Sublinhado */
text-decoration-line: overline; /* Linha acima */
text-decoration-line: line-through; /* Riscado */
text-decoration-line: none; /* Remove a decoração */
```



-------------














# 2. `text-decoration-color` Define a **cor da linha que aparece no texto**.

```css
text-decoration-color: blue;
text-decoration-color: #FF0000; /* vermelho em hexadecimal */
```




















# 3.`text-decoration-style` Define o **estilo da linha** que aparece no texto.

```css
text-decoration-style: solid;    /* linha reta padrão */
text-decoration-style: double;   /* duas linhas */
text-decoration-style: dotted;   /* pontilhada */
text-decoration-style: dashed;   /* tracejada */
text-decoration-style: wavy;     /* ondulada */
```





















# 4. `text-decoration-thickness`define a **espessura** da linha que aparece no texto.

```css
text-decoration-thickness: 2px;
text-decoration-thickness: thick;
```
