**`progress`** é um elemento usado para **exibir visualmente o progresso de uma tarefa**.

Exemplo:

```html
<label for="download">Download:</label>
<progress id="download" value="50" max="100">50%</progress>
```

**Atributos principais:**

- **Value** é o atributo do progress que define o valor atual do progresso.
    
- **Max** é o atributo do progress que define o valor máximo do progresso.
