# **`input type="checkbox"`** é um campo de entrada usado para **selecionar uma ou mais opções**.

Exemplo:

```html
<form>
  <label>
    <input type="checkbox" name="interesses" value="musica"> Música
  </label>
  <label>
    <input type="checkbox" name="interesses" value="esporte"> Esporte
  </label>
</form>
```
