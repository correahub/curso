# **`input type="radio"`** é um campo de entrada usado para **selecionar apenas uma opção entre várias**.

Exemplo:

```html
<form>
  <label>
    <input type="radio" name="genero" value="masculino"> Masculino
  </label>
  <label>
    <input type="radio" name="genero" value="feminino"> Feminino
  </label>
</form>
```
