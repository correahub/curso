# **`label`** é um elemento usado para **associar texto descritivo a um campo de formulário**.

Exemplo:

```html
<form>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
</form>
```

**Atributos principais:**

- **For** é o atributo do label que define o `id` do elemento de formulário ao qual ele está associado.
