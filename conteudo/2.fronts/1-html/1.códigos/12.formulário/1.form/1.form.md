
# **`form`** é um elemento usado para **criar um formulário para enviar dados**.

Exemplo:

```html
<form action="/enviar" method="post" enctype="multipart/form-data" autocomplete="on">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome">
  <button type="submit">Enviar</button>
</form>
```

**Atributos principais:**

- **Action** é o atributo do form que define a URL para onde os dados serão enviados.
    
- **Method** é o atributo do form que define o método de envio (`get` ou `post`).
    
- **Enctype** é o atributo do form que define como os dados serão codificados (`application/x-www-form-urlencoded`, `multipart/form-data`, etc.).
    
- **Autocomplete** é o atributo do form que ativa ou desativa o preenchimento automático pelo navegador (`on` ou `off`).
