# **`datalist`** é um elemento usado para **fornecer uma lista de sugestões para um `<input list>`**.

Exemplo:

```html
<form>
  <label for="fruta">Escolha uma fruta:</label>
  <input list="frutas" id="fruta" name="fruta">
  <datalist id="frutas">
    <option value="Maçã">
    <option value="Banana">
    <option value="Laranja">
  </datalist>
</form>
```
