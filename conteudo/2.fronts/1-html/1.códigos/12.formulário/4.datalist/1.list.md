# **`input list`** é um campo de entrada usado para **fornecer sugestões de valores usando um `<datalist>`**.

Exemplo:

```html
<form>
  <label for="fruta">Escolha uma fruta:</label>
  <input list="frutas" id="fruta" name="fruta">
  <datalist id="frutas">
    <option value="Maçã">
    <option value="Banana">
    <option value="Laranja">
  </datalist>
</form>
```
