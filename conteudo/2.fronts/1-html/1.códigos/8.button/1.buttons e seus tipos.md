# **`button`** é um elemento usado para **criar um botão clicável**.

Exemplo:

```html
<form>
  <button type="submit">Enviar</button>
  <button type="reset">Resetar</button>
  <button type="button">Clique aqui</button>
</form>
```

- **Submit** é o valor do atributo button que **envia os dados do formulário**.
    
- **Reset** é o valor do atributo button que **limpa todos os campos do formulário**.
    
- **Button** é o valor do atributo button que **não tem ação padrão**, servindo para scripts ou interatividade personalizada.
