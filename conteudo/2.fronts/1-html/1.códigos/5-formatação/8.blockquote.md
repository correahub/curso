# **`blockquote`** é um elemento usado para **representar uma citação longa de outra fonte**, com o atributo cite dá para indicar para o navegador de onde veio aquela citação.

Exemplo:

```html
<blockquote cite="https://exemplo.com/artigo">
  O conhecimento é a chave para a liberdade.
</blockquote>
```
