# **`q`** é um elemento usado para **representar uma citação curta dentro do texto**, com o atributo cite dá para indicar para o navegador de onde veio aquela citação.

Exemplo:

```html
<p cite="https://exemplo.com/artigo">Ele disse <q>o conhecimento é poder</q> durante a aula. <span> Emanuel</span>
</p>
```
