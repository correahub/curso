# **`aside`** é um elemento usado para **representar conteúdo secundário ou complementar** ao conteúdo principal.

Exemplo:

**Dentro de um artigo:**

```html
<article>
  <h2>Como Plantar uma Árvore</h2>
  <p>Passo 1: Escolha o local...</p>

  <aside>
    <h3>Dica Rápida</h3>
    <p>Molhe o buraco antes de plantar para ajudar a raiz.</p>
  </aside>

  <p>Passo 2: Enterre a muda...</p>
</article>
```

**Fora do artigo (como uma barra lateral da página):**

```html
<main>
  <article>...conteúdo principal...</article>
</main>

<aside>
  <h3>Artigos Relacionados</h3>
  <ul>
    <li><a href="#">Como cuidar do jardim</a></li>
    <li><a href="#">As melhores ferramentas</a></li>
  </ul>
</aside>
```
