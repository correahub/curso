# **`article`** é um elemento usado para **representar um conteúdo independente e reutilizável**.

Exemplo:

```html
<main>

  <article>
    <h2>Como Plantar Feijões</h2>
    <p>Primeiro você precisa de algodão, água e alguns grãos de feijão...</p>
  </article>

  <article>
    <h2>História: O Cachorro Perdido</h2>
    <p>Era uma vez um cachorrinho que se perdeu na floresta...</p>
  </article>

</main>
```
