# **`section`** é um elemento usado para **agrupar conteúdos relacionados dentro da página**.

Exemplo:

```html
<main>
  <section>
    <h2>Sobre Mim</h2>
    <p>Eu gosto de desenhar, jogar videogame e programar.</p>
  </section>

  <section>
    <h2>Meus Projetos</h2>
    <p>Aqui estão algumas coisas legais que já fiz.</p>
  </section>
</main>
```
