# **`dl`** é um elemento usado para **criar uma lista de definição.** 

Exemplo:

```html
<dl>
  <dt>HTML</dt>
  <dd>Linguagem de marcação para a web</dd>

  <dt>CSS</dt>
  <dd>Linguagem de estilos</dd>
</dl>
```
