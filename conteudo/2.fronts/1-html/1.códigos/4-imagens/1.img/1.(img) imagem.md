# **`img`** é um elemento usado para **exibir uma imagem na página, através dos atributos src que mostra o caminho e alt que descreve a imagem.**

Exemplo:

```html
<img src="imagem.jpg" alt="Descrição da imagem">
```

-  **O Atributo html src é  usado para apontar o arquivo de um recurso externo que faz parte do conteúdo da página.**

```html
<img src="foto.jpg">
<script src="app.js"></script>
```

---
- **`alt`**: texto alternativo exibido se a imagem não carregar e usado por leitores de tela

<img src="imagem.jpg" alt="Descrição da imagem">
