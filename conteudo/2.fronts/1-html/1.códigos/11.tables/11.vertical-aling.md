# **`vertical-align`** é uma **um atributo** usado para **alinhar o conteúdo de uma célula ou elemento verticalmente** (topo, meio ou base).

Exemplo:

```html
<table border="1">
  <tr>
    <td style="vertical-align: top;">Topo</td>
    <td style="vertical-align: middle;">Meio</td>
    <td style="vertical-align: bottom;">Base</td>
  </tr>
</table>
```
