# **`<a>` é um elemento usado para criar hiperlinks, que permite ao usuário navegar por entre as páginas de um site.** 

- `href` é o atributo que define para onde o usuário será direcionado ao clicar no hiperlink.
	-  usar #ID para hiperlinks de rolagem.
    
- `target` é o atributo que define como o hiperlink ira funcionar ao ser clicado pelo usuário. Podendo abrir o hiperlink e uma nova página com o valor `_blank`. 
    
- **`rel`** é um atributo  que define a **relação entre o hiperlink e a página atual**.
	- **`noopener`** é um valor do atributo rel que impede que a nova aba controle a página de origem.
	    
	- **`noreferrer`** é um valor do atributo rel que não envia o endereço da página de origem.
