**Complete para memorizar — Regras da Potenciação e Sinais**

| Nº  | Regra                                                                                         |
| --- | --------------------------------------------------------------------------------------------- |
| 1   | Base elevada ao expoente = potência.                                                          |
| 2   | Base elevado ao expoente 1 = Base                                                             |
| 3   | Base elevado ao expoente 0 = potencia 1, Se a base for diferente de 0                         |
| 4   | Base negativa elevada ao expoente **par** = potência **positiva**.                            |
| 5   | Base negativa elevada ao expoente **ímpar** = potência **negativa**.                          |
| 6   | Base negativa **dentro de parêntese**, elevada ao expoente **par** = potência **positiva**.   |
| 7   | Base negativa **dentro de parêntese**, elevada ao expoente **ímpar** = potência **negativa**. |
