
## Conjuntos numéricos são grupos de números que possuem características iguais.



### Por que isso é útil?

Porque assim fica mais fácil estudar e entender os números de acordo com onde eles são usados.

### Principais conjuntos numéricos:

| Nome do conjunto | Símbolo | O que inclui                                       | Exemplo                           |
| ---------------- | ------- | -------------------------------------------------- | --------------------------------- |
| **Naturais**     | **ℕ**   | Números usados para contar                         | 0, 1, 2, 3, 4…                    |
| **Inteiros**     | **ℤ**   | Naturais + negativos                               | -3, -2, -1, 0, 1, 2…              |
| **Racionais**    | **ℚ**   | Frações (e inteiros também)                        | 1/2, 3/4, -5, 2…                  |
| **Irracionais**  | **I**   | Números que **não** podem ser escritos como fração | π, √2, √3…                        |
| **Reais**        | **ℝ**   | Todos os racionais + irracionais                   | Todos os números da reta numérica |

---

### Explicando de forma simples:

Imagine uma **caixa grande** cheia de números (os **Reais**).  
Dentro dessa caixa, existem **outras caixas menores**, separando os números por tipo:

```
         Reais (ℝ)
     ┌────────────────┐
     │   Racionais (ℚ)│
     │  ┌────────────┐│
     │  │Inteiros (ℤ)││
     │  │┌──────────┐││
     │  ││Naturais (ℕ)││
     │  │└──────────┘││
     │  └────────────┘│
     └────────────────┘
     + Irracionais (I) (lado de fora dos racionais)
```

---

### Resumo:

**Conjuntos numéricos são formas de classificar os números conforme suas propriedades**, para facilitar o estudo da matemática.
